# Multimodal AI Integrated Breast Cancer Diagnosis System

This repository contains a demo Next.js website for our project. The site lives in the `web` directory and can be run locally or deployed to Vercel. The demo includes a simple greeting form and a separate features page.

## Running Locally

```bash
cd web
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploying to Vercel

1. Create an account at [Vercel](https://vercel.com/) if you do not have one.
2. Install the Vercel CLI:
   ```bash
   npm i -g vercel
   ```
3. From the `web` directory, run `vercel` and follow the prompts to deploy your site.

The site uses a simple form to greet visitors and demonstrates key features of the research project.
