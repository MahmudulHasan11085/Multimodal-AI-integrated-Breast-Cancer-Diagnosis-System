# Multimodal AI Integrated Breast Cancer Diagnosis System

This repository contains a demo Next.js website for our project. You can run it locally or deploy it to Vercel. The site includes a simple greeting form and a separate features page.
## Running Locally

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploying to Vercel

1. Create an account at [Vercel](https://vercel.com/) if you do not have one.
2. Install the Vercel CLI:
   ```bash
   npm i -g vercel
   ```
3. From the project root, run `vercel` and follow the prompts to deploy your site.

The site uses a simple form to greet visitors and demonstrates key features of the research project.
